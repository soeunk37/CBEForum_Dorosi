# CBEForum_Dorosi
2022 중앙대학교 경영경제대학 학술제 대상 수상


## 개요
* 주제 : 이륜차 사고다발구역 예측
* 기간 : 2022.04. ~ 2021.05.
* 팀명 : 도로시
* 팀원 : 김서린, 김소은, 서진슬, 장효정
* 사용언어 및 tool : Python, QGIS

* Correlation
* K-means Clustering  
2. GIS 분석
